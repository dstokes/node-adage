node-adage
==========

Fortune (Unix) for Node

example
=======
```js
var adage = require('adage');
var options = {};

adage(options, function(err, a) {
  console.log(a);
}); // => "You will always get the greatest recognition for the job you least like."
```

install
=======

With [npm](http://npmjs.org) do:

```
npm install adage
```
